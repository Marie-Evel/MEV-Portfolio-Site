My Work

StreetCred |

urban scavenger hunt app

StreetCred is a native app that takes sightseeing to a whole new level! It provides scavenger hunts as a fun way for tourists to explore U.S. cities. The app was the main project for an immersive UX designer certification course I took at CareerFoundry. I was responsible for the entire UX process from start to finish.

View case study

My Word! |

flashcard learning app

There's so much to learn! My Word is an app that allows students and learners of all ages to create and organize flashcards. It includes a study mode to test concepts learned and track progress. That was my very first UX design project, in which I learned about user research, user personas, flowcharting, task analysis, low-fidelity wireframing & prototyping, and usability testing.
